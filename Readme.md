# Microcontroller Based Digital Logic IC and Transistor Tester

## Project Overview
This project, **Design and Development of a Microcontroller Based Digital Logic Integrated Circuit and Transistor Tester**, was developed as part of the requirements for the **Bachelor of Science in Electronics and Communication Engineering** at the **Institute of Science and Technology, National University Bangladesh**.

The device is designed to test **digital logic ICs, transistors, and other common components** efficiently. It provides a cost-effective, user-friendly, and computer-independent solution for debugging electronic circuits.


## Features
- **Digital Logic IC Testing** using truth table matching method.  
- **Transistor & Other Common Component Testing** using short-circuit detection method.   
- **LCD Display Output** for real-time test results.  


## Tools & Technologies
- **Microcontroller:** Arduino Uno/Nano (*ATmega328P*)  
- **Programming Environment:** Arduino IDE    
- **Display:** I2C 16X2 LCD Module  


## Methodology
1. **IC Testing:**  
   - Input signals are applied to the IC.  
   - Output is compared with the expected truth table.  
   - Results are displayed on LCD.  

2. **Transistor and Other Components Testing:**  
   - Short-circuit detection method is used.  
   - Identifies open or short terminals.  
   - Displays status on LCD.
